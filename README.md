## Installation

1. Setup [React-Native](https://reactnative.dev/docs/environment-setup).
2. Clone the repo:

```
git clone https://github.com/shaunsaker/nou-braai.git
```

3. Install dependencies:

```
cd ./nou-braai && yarn
```

## Running

### iOS

```
yarn ios
```

### Android

```
yarn android
```
